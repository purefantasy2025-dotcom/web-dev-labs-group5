# Week 1: HTML Document Structure

## Objective
The objective of this lab session was to reinforce HTML fundamentals by creating a structured, multi-page website. This included implementing proper document structure, navigation, and multimedia elements.

## What Was Done
In this lab, we completed the following tasks:
* Created a properly structured HTML document using `<!DOCTYPE html>`, `<html>`, `<head>`, and `<body>` tags.
* Included essential meta tags and page titles.
* Developed a 4-page website consisting of Home, About, Contact, and Link to External pages.
* Implemented functional internal navigation links between all pages.
* Integrated multimedia by adding one local image and one online image from Unsplash.
* Organized content using headings, paragraphs, and lists.

## Challenges Faced
* Linking the local image correctly relative to the folder structure.
* Ensuring the navigation links worked across all four sub-pages.
